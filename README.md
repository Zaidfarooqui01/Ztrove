# Ztrove Android Application

## Overview

Ztrove is a feature-rich Android application developed using **Android Studio** that demonstrates secure authentication, real-time data handling, and advanced usage of mobile device sensors and system features. The app integrates multiple utility features into a single platform while maintaining data security and performance.

## Key Features

### 1. Secure Authentication

* Firebase **Email & Password Authentication**
* Ensures secure user login and account management

### 2. Real-Time Database Integration

* User information is stored in **Firebase Realtime Database**
* Data is organized in structured **nodes** for scalability and easy access

### 3. Multi-Feature Utility Modules

The application integrates several mini-app–like functionalities, including:

* **Voice Calculator**
  Allows users to perform calculations using voice commands.

* **Theft Protection System**
  Uses the **accelerometer sensor** to detect device movement and trigger alerts for unauthorized access.

* **Flashlight & Camera on Rotation**
  Automatically activates the flashlight and camera based on device orientation and rotation.

* **Video Capture**
  Enables users to record videos directly from within the app.

* **Text-to-Speech (TTS)**
  Converts text input into audible speech using Android’s Text-to-Speech engine.

* **Accelerometer Demonstrations**
  Multiple features showcasing real-time accelerometer data usage for motion detection and interaction.

## Technology Stack

* **Language:** Java (Android)
* **IDE:** Android Studio
* **Backend Services:** Firebase

  * Firebase Authentication
  * Firebase Realtime Database
* **Sensors Used:**

  * Accelerometer
* **Android APIs:**

  * Camera API
  * Flashlight API
  * Text-to-Speech API
  * Media Recorder

## Security

* User data stored safely in Firebase Realtime Database
* Access controlled via authentication-based database rules
